## O que são Estruturas de Dados?
Estruturas de dados são formas de organizar e armazenar dados em um computador de modo que possam ser acessados e modificados de forma eficiente. A escolha da estrutura correta pode impactar drasticamente o desempenho de um algoritmo (tempo de execução e consumo de memória).

Algumas das estruturas que vamos explorar aqui incluem:
* **Pilhas (Stacks):** LIFO (Last In, First Out)
* **Filas (Queues):** FIFO (First In, First Out)
* **Listas Encadeadas (Linked Lists):** Simples e Duplamente encadeadas
* **Tabelas Hash (Hash Tables):** Mapeamento de chaves para valores
